Must Know Cold

Multiplication Tables · 2 to 20

21–30 shortcut: You don't memorise 23×7. You do 20×7 + 3×7 = 140 + 21 = 161. Train this split mentally — it's faster than recall for big tables.
Essential

Squares · 1² to 35²

Pattern anchor: n² = (n−1)² + (2n−1). So 18² = 17² + 35 = 289 + 35 = 324. Use this to reconstruct if you blank.
Ending digits pattern: squares end in 0,1,4,5,6,9 — never 2,3,7,8. Instant check.
Essential

Cubes · 1³ to 15³

Why cubes matter: Number series, DI, and simplification questions use cube roots. Know 1–15 cold.
Non-Negotiable

Fraction → Percentage Equivalents

The golden rule: Never multiply by 0.1667 — instead think "1/6". All % calculations are fraction-first. Build from 10%: find 10%, halve it for 5%, quarter it for 2.5%, add combinations.

How to drill tonight

The 90-Minute Method

Round 1 — Read (20 min): Scan every table, square, cube, fraction on this page. Don't try to memorise yet. Just get familiar. Two passes.

Round 2 — Cover & Recall (30 min): Cover the answers. Say the value out loud before uncovering. Go through squares first (fastest ROI), then tables 2–12, then fractions.

Round 3 — Weak spots only (20 min): Write down every value you blanked on. These are your weak spots. Drill only those 10 more times.

Round 4 — Speed test (15 min): Random quiz yourself. Say the answer in under 2 seconds. If it takes longer, it's not memorised yet — it's calculated. Memorised = instant.

Target: Tables 2–12 = instant (<1s)
Target: Tables 13–20 = fast (<2s)
Target: Squares 1–25 = instant
Target: Squares 26–35 = fast (<3s)
Target: All fractions = instant

Sleep after. Memory consolidates during sleep — this is not optional.
